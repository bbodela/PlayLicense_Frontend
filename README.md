## 배포 방법

- origin코드를 clone 후 작업 시 branch에서 작업합니다.
- 코드를 origin branch로 push합니다.
- Pull Request를 합니다.
- Vercel이 자동으로 error가 있는지 체크해줍니다.
- error나 conflict 코드를 수정 후 `master`로 merge합니다.
- 배포 끝✨️
